### Installation
You can install the necessary dependecies by following steps bellow:

```sh
$ git clone https://github.com/MuhiaKevin/todo-list.git
$ cd todo-list
$ npm install
```


### Running the program
Execute the following command to run the app

Example :
```sh
$ npm start
```
